# Bundled colour map

This directory contains the RGB table for the **Oslo** colour map from the Scientific Colour Maps collection by Fabio Crameri.

The file is included to make the visualization workflow reproducible and self-contained.

Original resource:

- Fabio Crameri, Scientific Colour Maps
- https://www.fabiocrameri.ch/colourmaps/
